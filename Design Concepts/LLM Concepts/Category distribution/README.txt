# Project Name

Collaboration between two LLMs for design concept generation.

## File Description

This excel file "Category List & Descriptions-distribution" contains 
two sheets for two design problems. Each sheet organizes concept labels into categories (bins).
Each row represents a category, and the first twelve column contains the labels
corresponding to concepts assigned to that category.

Bin name: The category name.
Description (optional): Description of each category.

# Original Concept Distributions

- GPT-5.2: Labels for concepts generated by GPT-5.2, grouped under their corresponding categories.
- Claude Sonnet 4.5: Labels for concepts generated by Claude Sonnet 4.5, grouped under their corresponding categories.
- Llama 3.2: Labels for concepts generated by Llama 3.2, grouped under their corresponding categories.

# Refined Concept Distributions
The following columns contain solution labels for concepts generated by one model and refined by another. 
Labels are grouped under their corresponding categories:

- GPT generated-GPT refined
- Claude generated-GPT refined
- Llama generated-GPT refined
- GPT generated-Claude refined
- Claude generated-Claude refined
- Llama generated-Claude refined
- GPT generated-Llama refined
- Claude generated-Llama refined
- Llama generated-Llama refined

# Concept Counts
These columns report the total number of concepts for each model or model pair:

- protected.gpt-5.2 number: Total number of concepts generated by GPT-5.2 in corresponding category.
- protected.Claude Sonnet 4.5 number: Total number of concepts generated by Claude Sonnet 4.5 in corresponding category.
- protected.llama3.2 number:  Total number of concepts generated by Llama 3.2 in corresponding category.
- GPT generated-GPT refined number: Total number of concepts generated and refined by GPT-5.2 in corresponding category.

- Claude generated-GPT refined number: Total number of concepts generated by Claude and refined by GPT-5.2 in corresponding category.
- Llama generated-GPT refined number: Total number of concepts generated by Llama and refined by GPT-5.2 in corresponding category.
- GPT generated-Claude refined number: Total number of concepts generated by GPT and refined by Claude in corresponding category.
- Claude generated-Claude refined number: Total number of concepts generated by Claude and refined by Claude in corresponding category.
- Llama generated-Claude refined number: Total number of concepts generated by Llama and refined by Claude in corresponding category.
- GPT generated-Llama refined number: Total number of concepts generated by GPT and refined by Llama in corresponding category.
- Claude generated-Llama refined number: Total number of concepts generated by Claude and refined by Llama in corresponding category.
- Llama generated-Llama refined number: Total number of concepts generated by Llama and refined by Llama in corresponding category.

- Total number of concepts in each category: Total number of solution labels listed in each category (row).
- Average novelty: The calculated novelty score for each category
