# Project Name

Collaboration between two LLMs for design concept generation.

## File Description

The folder "Corn shucker" contains four excel files below:

"Corn_shucker_original_solutions": 
This file contains the original concepts generated by GPT 5.2, Claude Sonnet 4.5, and Llama 3.2, each in a separate sheet, for the corn shucker problem.

Each sheet includes the following columns:
- Solution title  
- Solution label (used for categorization)  
- Detailed steps of the concept  
- Explanations of the concept  
- Quality score (based on a three-point scale)  
- Generator model (indicates which LLM produced the original concept)

"Corn refined_Claude": 
This file contains concepts originally generated by GPT 5.2, Claude Sonnet 4.5, and Llama 3.2, and then refined by Claude.

The structure is similar to the original solutions file, with an additional column:
- Refiner model

Sheets are named using an originator–refiner format (e.g., GPT–Claude), where the first model generated the concept and the second refined it.
All sheets in this file have Claude as the refiner.


"Corn_refined_GPT": 
This file contains concepts originally generated by GPT 5.2, Claude Sonnet 4.5, and Llama 3.2, and then refined by GPT.

The columns are the same as in the original solutions file, with an added:
- Refiner model

Sheet names follow the originator–refiner format. All sheets in this file have GPT as the refiner.


"Corn refined_Llama":
This file contains concepts originally generated by GPT 5.2, Claude Sonnet 4.5, and Llama 3.2, and then refined by Llama.

The columns match the original solutions file, with an additional:
- Refiner model

Sheet names follow the originator–refiner format. All sheets in this file have Llama as the refiner.
