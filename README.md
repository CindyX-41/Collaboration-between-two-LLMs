# Collaboration-between-two-LLMs

## Dataset Description

This repository contains datasets for the analysis of design concept generation by large language models (LLMs).

### LLM-Generated Concepts
Design concepts were generated using:
- GPT-5.2  
- Claude Sonnet 4.5  
- Llama 3.2  

For each design problem, the dataset includes:
- Initial concepts produced by each model  
- Refined concepts produced by each model  

### Category List & Descriptions-distribution
This file includes:
- A list of categories for each problem  
- Concept assignments
- Distribution data showing the distribution of concept categories broken down by model and by phase (initial vs. refinement)

### Human Concept Data
A separate file contains category labels identified from human-generated design concepts based on the same design problems. The raw human concepts are not included to preserve confidentiality. This dataset serves as a baseline for comparison with LLM-generated outputs in future work.
